About Me

Welcome. I’m an academic researcher (currently a reader/associate professor) in machine learning. If you like to learn more about my research, please see my professional web page.

I write here about aspects of academic life that aren’t my research. I try to write as an essayist, to attempt to figure things out, how the University works, how to be a better researcher, how to give better advice to students. Well, except when, sometimes, I write just to be silly.
